# Launch Checklist

- Replace placeholder values in `.env`
- Confirm live contact email
- Confirm live booking URL
- Connect `PUBLIC_CONTACT_FORM_ENDPOINT`
- Add live analytics IDs if tracking is required at launch
- Add GitHub repository variables for the live contact details if deploying through Pages
- Add a real pilot demo or case study if one is ready for public use
- Finalise privacy, cookie, and terms wording after legal review
- Confirm canonical site URL for production
- Run `npm run build`
- Run `npm run check`
- Deploy `dist/` or connect the project to the chosen static hosting platform
